# Economics-of-Energy-Markets

## Physical and Monetary Flows between France and its neighbors
* Cross-border Physical Flows intra-day and yearly
* Monetary Flow from country x to y = Physical Flow * Energy price in x
* Net flows
* Focus on 2022 crisis

## Developping new transmission capacities
* Utilization rate = Flow / Transfer Capacity

If the Flow is much lower than the Transfer Capacity, no need to invest,
if it is very close, the border is saturated and there is a need to invest.

* Congestion rent = Spread × Flow with Spread between countries x and y = Price_x - Price_y.

If the Spread is high and the Flow is maximum, then there is a lack of capacity
and therefore a need to invest.
